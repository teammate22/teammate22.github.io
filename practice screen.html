Электронное учебное пособие по веб разработке
Учебный материал
HTML и CSS:
  • Создайте простую веб-страницу, содержащую заголовок, текстовый блок и изображение. Оформите ее с помощью CSS, добавив стили для текста, фона и изображения.
  • Используя HTML формы, создайте страницу с несколькими элементами ввода, такими как текстовое поле, кнопка и переключатель. С помощью CSS стилизуйте форму для улучшения ее внешнего вида.

JavaScript:
  • Напишите функцию JavaScript для вычисления факториала числа.
  • Создайте интерактивную страницу, на которой при клике на кнопку отображается текущая дата и время.

Работа с событиями:
  • Напишите код JavaScript для отслеживания события нажатия клавиши и вывода названия нажатой клавиши.
  • Создайте функцию JavaScript, которая будет обрабатывать событие щелчка мыши и изменять стиль элемента на странице при каждом клике.

Функции:
  • Напишите функцию JavaScript для поиска наибольшего числа в массиве.
  • Создайте анонимную функцию в JavaScript, которая принимает два аргумента и возвращает их сумму.

Работа с объектами:
  • Создайте объект JavaScript, представляющий информацию о пользователе (имя, возраст, email и т. д.). Напишите функцию для вывода этой информации на страницу.
  • Напишите функцию JavaScript для проверки, является ли данное значение объектом или нет.

Работа с массивами:
  • Напишите функцию JavaScript для объединения двух массивов и удаления дубликатов.
  • Создайте массив объектов, представляющих различные товары в интернет-магазине. Напишите функцию для фильтрации товаров по категории или ценовому диапазону.

Работа с формами:
  • Создайте HTML-форму для отправки сообщения с полями для имени, email и сообщения. Напишите код JavaScript для валидации введенных данных перед отправкой формы.
  • Напишите функцию JavaScript для очистки полей формы после успешной отправки данных на сервер.

Работа с cookie:
  • Используя JavaScript, создайте функцию для установки cookie с определенным значением и сроком действия.
  • Напишите код JavaScript для чтения значения cookie и отображения его на веб-странице.
